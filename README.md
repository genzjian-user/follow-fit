# 跟着减 - 有氧减脂语音私教APP

## 项目概述
基于Flutter开发的跨平台健身应用，提供个性化有氧减脂计划、语音指导、GPS运动跟踪、蓝牙心率监测和AI聊天功能。

## 核心功能模块

### 1. 语音播报模块
- 训练语音提示和指导
- 动作计数语音反馈
- 休息时间提醒
- 完成进度报告

### 2. GPS运动记录模块
- 实时轨迹记录
- 距离、速度、海拔计算
- 运动数据统计
- 路线地图展示

### 3. 蓝牙心率模块
- 蓝牙BLE设备连接
- 实时心率数据显示
- 心率区间提醒
- 心率历史图表

### 4. AI隔离模块
- AI健身问答
- 个性化建议
- 运动数据分析
- 饮食推荐

### 5. 运动历史模块
- 本地数据存储
- 历史记录查看
- 进步趋势分析
- 数据导出功能

## 技术架构
- **框架**: Flutter 3.x
- **状态管理**: Riverpod
- **导航**: Go Router
- **数据库**: Hive (本地存储)
- **地图**: Flutter Map / Google Maps
- **蓝牙**: Flutter Blue Plus
- **语音**: TTS (Text-to-Speech)
- **AI**: OpenAI API / 本地模型
